# TimeTable

**語言：** [English](README.md) · [简体中文](README.zh-Hans.md) · 繁體中文

TimeTable 是一款面向學生、自律族、備考與高效工作者的「時間表匯入 & 編輯工具」。
核心目標：把「別人給的時間表」或「自己寫的安排」快速變成可用的日程列表，並支援編輯、提醒、匯出與範本復用。

## 主要優點

- 匯入即用：從 Markdown 表格或逐行文字一鍵解析成時間表。
- 手動也快：逐行輸入與卡片編輯兼顧速度與精細度。
- 解析智能：支援多種時間分隔符、時長格式與 emoji 辨識。
- 高容錯：錯誤列高亮提示，快速定位修正。
- 自動補齊：缺少結束時間可用下一條開始時間自動補齊。
- 跨天支援：晚間到次日的時間段可正確標示。
- 範本復用：常用日程儲存為範本，一鍵套用。
- 清晰預覽：匯入後先預覽，確認無誤再匯入。
- 本地離線：不需網路，隱私安全，效能穩定。
- iOS 體驗：行動端友好，隨手記錄不打斷節奏。

## 核心特點

- 多輸入格式
  - Markdown 表格匯入
  - 逐行文字匯入
  - 手動逐條編輯

- 統一資料結構
  - 開始時間、結束時間、標題、時長、備註、標籤、表情

- 關鍵規則
  - 時間衝突提示
  - 自動排序與補齊
  - 跨天辨識與顯示

- 範本能力
  - 新建、復用、快速套用

- 預覽與確認流程
  - 先預覽，再確認匯入

## 適用情境

- 朋友分享時間表，想 5 秒匯入。
- 備忘錄幾行文字，快速轉成時間塊。
- 每天結構相似，需要範本快速復用。

## MVP 範圍（已涵蓋）

- 逐行文字解析
- Markdown 表格解析
- 預覽與錯誤高亮
- 範本保存與套用
- 基礎編輯流程

## 未來可擴充方向

- 同步系統行事曆
